# OLA Ride Data Analysis

## Project Overview
This project presents an end-to-end analysis of OLA ride booking data with the objective of deriving actionable business insights.  
The analysis focuses on ride demand patterns, cancellations, customer behavior, and revenue trends to support data-driven decision-making.

## Tools & Technologies
- SQL – Data extraction, querying, and analysis
- Power BI – Interactive dashboards and visual analytics
- Python – Data manipulation and analysis
- Streamlit – Web-based data analytics application

## Key Insights
- Identification of peak booking hours and high-demand time periods
- Analysis of ride cancellation trends and contributing factors
- Revenue distribution across different ride categories
- City-wise and time-based ride demand patterns

## Project Structure
├── app.py
├── ola_data.csv
├── Ola_ride_SQL_analysis.sql

## How to Run the Streamlit App
1. Install required dependencies:
```bash
pip install streamlit pandas plotly

```
2. To run streamlit application
 ````bash
streamlit run app.py
````

4. Open Application in your browser
````bash   
http://localhost:8501
````


## Power BI Dashboard
The Power BI dashboard provides interactive insights into:
- Total rides and revenue trends
- Ride cancellation analysis
- Peak booking hours
- City-wise and ride-category performance

To view the dashboard:
- Open `Ola_ride_BI.pbix` using **Power BI Desktop**
- Interact with filters and visuals to explore insights
